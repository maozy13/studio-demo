 开发 DIP 项目管理 DEMO

## 要求
仔细阅读以下两个文档：
* 系统元数据：@design/DIP Studio.pdf
* 项目管理：@design/项目管理.pdf

- DIP 项目管理 DEMO 本身遵循这两个文档的项目架构描述
- 界面风格参考 Notion，保持简约并使用留白

## 技术要求
- 使用 TypeScript 作为开发语言
- 使用 Tailwind CSS 作为 CSS 框架
- 使用 ReactJS 作为前端开发框架
- 使用 Ant Design V6 作为 UI 组件，当涉及到组件选择、API 使用、Props 和 示例时，参考：@docs/vendor/antdesign/llms.txt
- 使用 Vite 作为构建工具
- 为每个类、函数，以及关键的判断条件加上注释
- 不要创建 README
